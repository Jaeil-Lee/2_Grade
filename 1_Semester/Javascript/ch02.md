### 2.3.3 불값 (boolean)
  - true/false
  - `숫자로 취급`하면 : `1(true)` or `0(false)` 으로 판단
  - 비교연산의 결과/논리연산의 결과 
  - ※ `false로 처리되는 경우`※
    - 0
    - undefined
    - null
    - NaN
    - `` : 빈문자열
  - ※ `true로 처리되는 경우`※
    - 상기 false에 해당하는 것 빼고 전부다


- NaN 과 NaN 비교
- null 과 null 비교
- undefined 비교 

- 문자열의 비교 
  - 문자에 대한 값: ascii값(unicode)
  - 'a'.`charCodeAt()` => 97  , `'abc'.charAt(2).char.charCodeAt()` => 98

- == 와 `===` 
  - 동일한점
    - 값을 비교
  - 차이나는 점
    - 데이터 `타입도 비교` 여부
      - ===타입을 비교한다

- != 와 !==  ( `! (exclamation)` => `not` )

#### 논리 연산자
- && : AND  ( AMPERSAND )
  - ~하고  
  - a && b
    - a의 값과 b의 값이 true인 경우만 true 결과 

<br />

- || : OR  ( BAR )
  - ~이거나
  - `a || b`
    - a의 값과 b의 값 중 어느 하나라도 true인 경우 true 결과
    - `a의 값과 b의 값이 모두 false인 경우 false 결과`
    - Enter 입력? || Tab 입력 ?

<br />

- !: NOT
  - ~이지 않다
  - 남자가 아니다.
  - 100이지 않다 
    - a = 100
    - !a
    - !! 는 해당 값을 boolean 값으로 반환한다
